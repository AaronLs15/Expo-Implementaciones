# Demo Mercado Pago Expo

Proyecto didactico para ensenar una integracion base de Mercado Pago Checkout Pro en Expo, enfocado en iOS Simulator con development build.

## Comandos principales

```bash
npm install
npm run server
npm start
npm run ios
```

## Documentacion completa

La guia paso a paso esta en:

- [docs/MERCADO_PAGO_EXPO_PASO_A_PASO.md](./docs/MERCADO_PAGO_EXPO_PASO_A_PASO.md)

## Variables de entorno

Copia `.env.example` a `.env` y completa tu Access Token de pruebas:

```bash
cp .env.example .env
```
